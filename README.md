# uist-portal
Demo website for UCEP Institute of Science &amp; Technology, Dhaka
🏛️ UIST Portal
Demo Website — UCEP Institute of Science & Technology, Dhaka

📌 Overview
UIST Portal is a fully custom-built, single-file website designed for UCEP Institute of Science & Technology (UIST) — one of Dhaka's leading BTEB-affiliated polytechnic institutes, established in 2015 under UCEP Bangladesh.
This project was built as a complete redesign — significantly more modern, interactive, and feature-rich than the original institutional website.

✨ Features

🎨 Engineering-themed dark design — Navy + Gold color scheme inspired by the UIST logo
🌐 Three.js 3D animations — Interactive hero canvas with rotating gear, DNA helix, floating particles, and grid
🧠 About Section 3D — Dodecahedron + orbiting icosahedrons visualizing the "knowledge core"
📄 6 Course Detail Pages — Each department has its own dedicated full-screen detail page with subjects, career paths, and admission requirements
💼 Job Placement Section — Animated progress bars showing placement statistics
📋 Admission Info — Full fee structure, eligibility criteria, and required documents
🎓 Student Testimonials — Real quotes from UIST students
📬 Contact Form — Admission inquiry form with department selection
👤 Developer Popup — 5-second intro popup with developer credit (auto-dismiss)
🔝 Back to Top — Animated scroll-to-top button
📱 Responsive — Mobile-friendly layout

### 🛠️ Tech Stack (Clean & Elegant Version)

**🔹 HTML5**
Provides the core structure and semantic foundation of the website.

**🔹 CSS3**
Handles styling, smooth animations, and fully responsive layouts for all devices.

**🔹 JavaScript (ES6)**
Adds dynamic interactivity, including scroll effects and animated counters.

**🔹 Three.js (r128)**
Enables immersive 3D WebGL animations, especially in the hero and about sections.

**🔹 Google Fonts**
Carefully selected fonts like *Orbitron*, *Rajdhani*, *Share Tech Mono*, and *Exo 2* to create a modern, tech-inspired visual identity.
